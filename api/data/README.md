## Dados da API

Este diretório guarda artefatos gerados pelo escopo SharePoint Graph.

## Estrutura

- `inventory/`
  - Saídas JSON geradas por validações ou futuras rotinas de inventário SharePoint.

## Boas práticas

- Não versionar dados reais de tenant.
- Arquivar resultados antigos fora do repositório.
- Remover qualquer artefato legado que não faça parte do fluxo SharePoint.
