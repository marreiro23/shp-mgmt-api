# Audit data store

This folder stores persisted audit trail events generated by governance
operations.

## Current storage model

- file-based JSON persistence
- append-only event records
- operationId and correlationId linkage

## Event usage

Audit records are used for:

- operational traceability
- troubleshooting and incident analysis
- export and compliance evidence

## Notes

This folder may contain runtime-generated data and should not be treated as a
source-code artifact.
